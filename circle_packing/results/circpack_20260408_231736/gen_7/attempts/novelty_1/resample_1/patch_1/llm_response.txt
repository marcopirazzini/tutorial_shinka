<DESCRIPTION>
Introduce a hexagonal lattice initialisation to improve initial center packing density, and add a lightweight second-start annealing pass to diversify the search and escape local optima. The hex-grid is generated locally around the square center using a triangular lattice basis, ensuring no overlaps near walls and better space-filling than two fixed rings. A second independent restart (with a different RNG seed) refines the arrangement further, potentially increasing the total sum of radii beyond prior runs. These changes are designed to be incremental and self-contained, preserving the existing interface while likely boosting performance without relying on external libraries.

    # --- Hex-grid initialisation for dense packing (embedded helper) ---
    def hex_grid_centers(n, center=(0.5, 0.5), s=0.08):
        cx, cy = center
        a1 = np.array([s, 0.0])
        a2 = np.array([s/2.0, (np.sqrt(3)/2.0) * s])
        pts = []
        K = 6
        for i in range(-K, K+1):
            for j in range(-K, K+1):
                off = i * a1 + j * a2
                pos = np.array([cx, cy]) + off
                if 0.01 <= pos[0] <= 0.99 and 0.01 <= pos[1] <= 0.99:
                    pts.append(pos)
        pts = np.array(pts)
        if pts.shape[0] == 0:
            return np.zeros((n, 2))
        dists = np.linalg.norm(pts - np.array([cx, cy]), axis=1)
        idx = np.argsort(dists)
        centers_hex = pts[idx, :][:n]
        if centers_hex.shape[0] < n:
            pad = np.zeros((n - centers_hex.shape[0], 2))
            centers_hex = np.vstack([centers_hex, pad])
        return centers_hex

    centers = hex_grid_centers(n, center=(0.5, 0.5), s=0.08)
    centers = np.clip(centers, 0.01, 0.99)

    radii = compute_max_radii(centers)
